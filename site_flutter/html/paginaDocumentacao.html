Construa uma aplicação web com Flutter

Requesitos:

  • Flutter SDK. link de instalação
  • Para depurar o aplicativo é necessário o navegador Chrome
  • Uma IDE, que pode ser opcional VScode, IntelliJ...

Siga os passos a seguir e em breve você estará com aplicativos inovadores e de qualidade


Configurar

Execute os seguintes comandos para usar a versão mais recente do Flutter SDK:

flutter channel stable

flutter upgrade

Se o Chrome estiver instalado, o flutter devicescomando gera um Chrome dispositivo que abre o navegador Chrome com seu aplicativo em execução e um Web Serverque fornece o URL que atende ao aplicativo.

flutter devices

1 connected device: Chrome (web) • chrome • web-javascript • Google Chrome 88.0.4324.150


Criar e executar

Para criar um novo aplicativo que inclua suporte web (além do suporte móvel), execute os seguintes comandos, substituindo my_app pelo nome do seu projeto:

flutter create my_app

cd my_app

Para vincular seu aplicativo localhostno Chrome, digite o seguinte na parte superior do pacote. O método run serve para iniializar a aplicação.

flutter run -d chrome


Construir

Gere uma compilação de seu aplicativo através do comando a baixo:

flutter build web

Você pode criar um build de lançamento usando o modo de lançamento ( flutter run --release) ou usando flutter build web.

Podem ser renderizados também com o html e css. --web-renderer html.
